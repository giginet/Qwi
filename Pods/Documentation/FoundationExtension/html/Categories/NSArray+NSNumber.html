indexOfInteger:

@brief Returns the lowest index whose corresponding array value is equal to a given integer.

- (NSUInteger)indexOfInteger:(NSInteger)value

Parameters

value

A value.

Return Value

The lowest index whose corresponding array value is equal to value. If none of the objects in the array is equal to NSNumber with value, returns [NSNotFound][NSNotFound].
