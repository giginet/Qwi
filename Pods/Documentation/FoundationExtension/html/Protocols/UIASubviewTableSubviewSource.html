Conforms to NSObject

Overview

@brief This protocol defines data source and subview source methods for UIASubviewTableView objects. @details This covers optional methods of data source, because table view self-implemented its required methods.

subviewTableView:viewHeightForRow:

@brief Asks the subview source the height of view for given row. @details This is used to avoid referencing real view for row to calculate height for the row. If not implemented, default value is view.frame.size.height for each view.

- (CGFloat)subviewTableView:(UIASubviewTableView *)scrollView viewHeightForRow:(NSUInteger)row

Discussion

@brief Asks the subview source the height of view for given row. @details This is used to avoid referencing real view for row to calculate height for the row. If not implemented, default value is view.frame.size.height for each view.

Warning: This doesn’t work if you implement tableView:heightForRowAtIndexPath:.
