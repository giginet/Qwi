imageWellShouldAcceptURLString:

@brief Asks the delegate if the image well should accept URL string.

- (BOOL)imageWellShouldAcceptURLString:(NSAImageWell *)imageWell

Parameters

imageWell

The image well which the receiver is the delegate.

Return Value

YES if the display controller should accepts string as URL of image; otherwise NO. @details Default is NO. If YES, the image well accepts string value as URL of image. This methods override image well’s acceptStringDragging value.

Discussion

@brief Asks the delegate if the image well should accept URL string.
